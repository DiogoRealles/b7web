O que é HTML5

HTML5 é uma tecnologia utilizada para estruturação e construção de páginas web.

O que são tags

Tags são como blocos que constroem as residências. HTML5 possui diversas tags para diversos usos: paragrafo: p, imagem: img, video: video. e muito mais.

Tag p

Tags são como blocos que constroem as residências. HTML5 possui diversas tags para diversos usos: paragrafo: p, imagem: img, video: video. e muito mais.

Tag "p" é utilizada para conteúdo textual.

<p> Esse conteúdo é um texto... </p>

Tag br

Tag "br" (break row) é utilizada para quebra de linha.

<br />

Listas

Listas são usadas para ordenar passos:

  1. Item 1
  2. Item 2
  3. Item 3

Podem ser sem numeração:

  • Item 1
  • Item 2
  • Item 3

Links

Links permite que seja possível navegar entre páginas e conteúdo em uma página HTML5.

Imagem

Tag img é utilizada para inserir imagens nas páginas web.

Tabela

Tabela é utilizada para mostrar dados tabulares como: classificação de times, lista de notas de alunos.

Formulários

Formulários é uma parte importante para o HTML5, é com ele principalmente que os dados são enviados para o back-end.
Formulário possui diversos campos: color, text, password, email, checkbox, radio, submit e muito mais.
