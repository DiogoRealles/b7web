Semana 01: HTML5 + CSS3

O que é HTML5:
É uma linguagem de marcação, ou seja, para estruturar as páginas web. Semelhante os blocos que estruturam e constroem as residências. HTML5 possui blocos/tags para vários usos como: paragrafos: "p", imagens: "img", video: "video", e muito mais.


O que são tags: Tags são como blocos que constroem as residências. HTML5 possui diversas tags para diversos usos: paragrafo: p, imagem: img, video: video. e muito mais.


Tag p: Tags são como blocos que constroem as residências. HTML5 possui diversas tags para diversos usos: paragrafo: p, imagem: img, video: video. e muito mais.

O que é CSS3:
É uma linguagem de estilização, ou seja, ajustar tamanho, cores e dar beleza as páginas web. Semelhante a pintura de uma residência. CSS3 permite estilizar textos, imagens, alinhar e posicionar elementos, animações e transformações e muito mais. .
